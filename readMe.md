## 信安二班毕业视频拍摄创意和想法。
    author : 信安二班全体.

一去二三年，转眼距离本科毕业就要三年了。
我们读研的这些个也要研究生毕业了，当然还有工作的各位，国外的各位，三年过去，都如何？

9月份旭哥来了趟成都，中间顺便向我交代了下拍摄我们3年之后的这个阶段的视频的想法，我就主要负责记录了。

Time flies.

这个project分享到github上，主要是考虑到大家群策群力，这些想法和创意会面临一改再改，所以放到这儿，大家可以都随时看到，
随时提出意见。大家可以说一下GitHub的账号或者邮箱，加进这个项目里。
